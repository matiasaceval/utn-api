The uri '/activity' and '/holiday' , indicates the next event closest to the current date

GET/ nextActivity : utn-api.herokuapp.com/activity
GET/ nextHoliday : utn-api.herokuapp.com/holiday
GET/ currentEvent : utn-api.herokuapp.com/current

The uri '/current' only indicates the activity or holiday on that date



All methods can receive date param like the next example

GET/ utn-api.herokuapp.com/activity?date=05/05/2022
